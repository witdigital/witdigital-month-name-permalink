== Description ==
Based off https://wordpress.org/plugins/month-name-permalink/ and (https://gist.github.com/rogerhub/8306875)
Updated for current WordPress version by Wit digital June 2021.

Plugin enables use of monthname (january, june) or monthcode (jan, jun) in permalinks

Supports permalinks in the form of /2014/nov/23/post-name or /2014/november/23/post-name


== Frequently Asked Questions ==

= How do i use it =

Navigate to Settings -> Permalinks

Select "Custom Structure" and enter the following structure

/%year%/%monthname%/%day%/%postname%/

Save the changes

== Installation ==

1. Upload to the plugins directory
2. Activate the plugin through the 'Plugins' menu in WordPress
